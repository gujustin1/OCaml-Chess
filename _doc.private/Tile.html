Module Tile

module Tile: sig .. end

Representation of the tiles and pieces on a chessboard

This module represents the state of each tile on the chessboard: whether there is a piece on it and if so, what kind, the colors of the pieces and a link to an img that that tile type.

type p =
| Empty
| Rook
| Bishop
| Knight
| Pawn
| Queen
| King

Types of pieces on a tile

type c =
| None
| White
| Black

Color of the piece on a tile

type img =
| Image of string
| No

img for a piece on a tile

type tile = {
   piece : p;
   color : c;
   mutable image : img;
}

Abstract type for the tiles on a chessboard
