Chessboard and all the functions on a board

Module represents the state of a chessboard and includes all functions that can be made on a board such as checking for checkmate, moving a piece, and initializing chessboards

type board = Tile.tile array array

The abstract type that represents a chessboard.

val possible_moves : int -> int -> board -> (int * int) list

possible_moves x y b is a set-like list of all the tiles that a piece at position (x,y) can move to.

val string_to_lists : string -> string list list

string_to_lists s is a string list list representation of s, where each string list is created by splitting on character '/' in s.

val starterboard : string

starterboard is the initial board state of a chess game as a string.

initialize b s 0 turns the string representation, s, of the board into a board of tiles, b.

val is_checkmate : board -> bool

is_checkmate b returns true if a king is in checkmate, and false otherwise.

val in_check : Tile.c -> bool

in_check c returns whether the c colored king is in check.

val check_validity : board -> int -> int -> int -> int -> int -> bool

check_validity b x y x2 y2 c is true iff moving (x,y) to (x2,y2) is a valid move in the state of the game. Examples of returning false includes moving another piece when the king is in check, moving the king into a checked square, and castling in failed conditions.

val move_piece : board -> int -> int -> int -> int -> bool

move_piece b x1 y1 x2 y2 moves the piece on b.(x1).(y1) to b.(x2).(y2). There is a precondition that the move is valid.
